# Tips

## Login to the H2 console
On your choice of browser, enter [http://localhost:{serverport}/h2-console]()
and use the following to login to the console:
```
  JDBC URL: jdbc:h2:mem:SEC
  User Name: sa
  Pasword: <leave this empty>
```
